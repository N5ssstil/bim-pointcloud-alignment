# BIM-点云施工质量检测系统

## 🎯 功能特性

自动化施工质量检测工具，支持：
- ✅ **楼层净高检测** - 实测vs设计值对比
- ✅ **房间尺寸检测** - 开间、进深测量  
- ✅ **墙面垂直度检测** - RANSAC算法
- ✅ **墙面平整度检测** - RMSE计算
- ✅ **自动报告生成** - TXT导出
- ✅ **GUI界面** - PyQt5/Tkinter双版本

---

## 🚀 快速开始

### 方案1：服务器环境（Tkinter版本）

```bash
cd /home/admin/.openclaw/workspace/bim-pointcloud-alignment

```

### 方案2：本地电脑（PyQt5版本）

**安装依赖：**

**运行：**

## 📊 检测标准

| 检测项目 | 合格标准 |
|----------|----------|
| 楼层净高偏差 | ≤50mm |
| 房间尺寸偏差 | ≤30mm |
| 墙面垂直度 | ≤3° |
| 墙面平整度(RMSE) | ≤8mm |

---

## 📝 使用步骤

1. **打开程序**
   ```bash
   python main_pyqt.py
   ```

2. **导入文件**
   - 点击"选择IFC文件" → 导入BIM模型
   - 点击"选择LAS文件" → 导入点云数据

3. **设置参数**
   - 平面检测阈值（默认50mm）
   - 最大检测平面数（默认10）

4. **开始分析**
   - 点击"开始分析"按钮
   - 等待分析完成

5. **查看结果**
   - 检测结果概览表格
   - 墙面质量详情
   - 详细报告文本

6. **导出报告**
   - 点击"导出报告"保存TXT文件
